### About

Improve your website's search engine optimization with SEO Meta Description AI, a WordPress plugin that harnesses the power of artificial intelligence to generate custom SEO meta descriptions for every post and page. With SEO Meta Description AI plugin, you can effortlessly optimize your content for search engines, captivate your audience, and drive more organic traffic to your site.

### Key features

1. **AI-Powered Meta Description Generation:** SEO Meta Description AI plugin utilizes cutting-edge AI technology to analyze your content and generate compelling meta descriptions that grab attention and improve search engine rankings.

2. **Easy Integration:** Seamlessly integrate the plugin into your WordPress site and take advantage of its intuitive user interface. No coding skills required!

3. **Customization Options:** Tailor your meta descriptions to match your unique content. Modify the generated descriptions, add specific keywords, or customize them entirely to align with your SEO strategy.

To ensure fair usage, the free version of the plugin has a global limit of 100 requests per day and an individual limit of 5 requests per day per IP address.

== Installation ==

### Installation from the WordPress admin dashboard

1. In the left-side menu select _Plugins > Add New_.

2. In the top-right corner select search by _Author_, and enter _binarysolutions_ as a search term.

3. Once the _SEO Meta Description AI_ plugin appears in the search results, click on _Install Now_, and then on _Activate_ button.

That's it! The plugin is now ready for use! You can also watch the installation and usage [instructional video](https://youtu.be/cHK4m0vL1WU).

### Manual installation

1. Unpack the archive in the WordPress plugins directory.

2. Activate the plugin through the dashboard.

That's all!

== Frequently Asked Questions ==

= How can I customize the meta description for a specific WordPress post or page? =

To customize the meta description for a specific post or page, simply edit the post or page in the WordPress admin console. After updating it, or saving the draft, click on `Replace with new SEO description` button inside the `SEO Meta Description AI` box. 

= What languages are supported for meta descriptions? =

The meta description will be generated in the language of your page or post content.

= Does the plugin support bulk editing of meta descriptions? =

No.
